# CMD+G — Ops & Legal TODO

Everything needed to operate as a legitimate, protected consultancy.
Use Google Docs / Notion / Lawyer to draft the actual documents — this is the checklist.

---

## 🔴 Legal (Do First)

- [ ] **Decide entity type** — LLC recommended (liability protection, pass-through tax)
  - Option A: One shared LLC (CMD+G LLC) — cleaner for clients, requires operating agreement
  - Option B: Each designer invoices independently — simpler, no shared entity needed
- [ ] **Operating Agreement** (if shared LLC) — covers profit split, decision-making, departure terms
- [ ] **EIN** — employer identification number from IRS (free, takes 10 min online)
- [ ] **Business bank account** — separate from personal. Required for LLC.
- [ ] **IP Assignment clause** — ensure client owns deliverables upon final payment (include in MSA)
- [ ] **NDA template** — mutual NDA for discovery calls. Short, plain-language.

---

## 🔴 Contracts

- [ ] **MSA (Master Services Agreement)** — the evergreen contract. Covers:
  - Scope of engagement
  - Payment terms (50% deposit, 50% on delivery — standard)
  - Revision rounds (recommend: 2 rounds included, additional billed hourly)
  - IP ownership transfer upon payment
  - Kill fee (25–50% if client cancels mid-project)
  - Confidentiality
  - Limitation of liability
- [ ] **SOW template (Statement of Work)** — attached per project. Covers:
  - Deliverables (specific, not vague)
  - Timeline + milestones
  - Project rate
  - What's out of scope
- [ ] **Invoice template** — consistent format. Include: project name, payment due date, bank/Venmo/wire details, late fee policy (1.5%/month)

---

## 🔴 Finance

- [ ] **Decide billing structure** — who invoices, how it splits
  - Cleanest: CMD+G invoices client → split 4 ways internally
  - Simplest: each designer invoices client for their portion (requires clear SOW split)
- [ ] **Set project minimums** — recommended starting floor: $5–10K/project
- [ ] **Set hourly rate** (for overages + ad-hoc) — senior design market rate: $150–250/hr
- [ ] **Payment processor** — Stripe (invoicing + ACH), Mercury (bank), or Wise (international)
- [ ] **Accounting** — track income/expenses from day one. Wave (free) or QuickBooks Simple.

---

## 🟡 Client Intake

- [ ] **Intake form** — short, qualifying. Fields:
  - Company / project name
  - What do you need designed?
  - Timeline
  - Budget range (give options: $5–10K / $10–25K / $25K+)
  - How did you hear about us?
- [ ] **Discovery call framework** — 30–45 min agenda:
  1. Their context (5 min)
  2. Project scope + constraints (15 min)
  3. How we work (5 min)
  4. Fit check — do we want this client? (internal, post-call)
- [ ] **Proposal template** — visual, on-brand. Cover: problem framing, our approach, deliverables, timeline, investment.
- [ ] **Onboarding checklist** — what client needs to send before kickoff:
  - Brand assets (if existing)
  - Stakeholder contacts
  - Access to tools (Figma, Notion, etc.)
  - Signed contract + deposit received
